What is BlackArch Linux?

BlackArch Linux is a lightweight expansion to Arch Linux for penetration testers.

Currently we have 626 tools in our toolset that continues to grow. All tools are thoroughly tested before being added to the codebase to maintain the quality of the repository. Each tool can be installed individually and does not require full installation of the distribution, to find out how to download the toolset see our instructions here.

This is a relatively new project and as we continue to grow the distribution with the help of the community, we continue to make it better. If you find bugs, have suggestions or have feature requests please feel free to visit our github repository or email us.
